About TSM

The THREDDS sewing machine (TSM) is a project to make the configuration of THREDDS data servers easier with a web based interface that lets you select data from Unidata's Internet Data Distribution (IDD) feeds. The tool emails users a zip file containing the necessary configuration files.

What is THREDDS?

The goal of Unidata's Thematic Real-time Environmental Distributed Data Services (THREDDS) is to provide students, educators and researchers with coherent access to a large collection of real-time and archived datasets from a variety of environmental data sources at a number of distributed server sites. The THREDDS Data Server (TDS) is a web server that provides metadata and data access for scientific datasets, using a variety of remote data access protocols.

Support

Having trouble with the TSM or setting up your THREDDS data server? We're here to help! Issues for the TSM may be filed on its GitHub repository. THREDDS questions should be directed to our support email at support-thredds@unidata.ucar.edu.
